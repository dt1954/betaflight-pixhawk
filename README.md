# Betaflight Firmware for Pixhawk-Style Flight Controllers

This project explores running **Betaflight firmware on Pixhawk-style flight controllers**, especially high-performance boards like **Kakute H7**.

The Kakute H7 hardware is powered by the **STM32H743 processor running at 480MHz**, and provides a powerful platform for FPV drones and experimental UAV development.

It includes:

- 6 UART ports
- MicroSD Blackbox logging
- Onboard OSD
- 3S–8S battery support
- High-speed H7 processor

## Hardware Tested

Currently tested on:

- Kakute H7
- H7-based flight controllers
- Multiple UART peripherals
- FPV drone setups

## Why Betaflight on Pixhawk Hardware?

Pixhawk boards are widely used in UAV research and development, while **Betaflight provides extremely fast control loops and powerful tuning capabilities for FPV drones**.

Combining these two ecosystems can enable:

- high performance FPV drones
- experimental UAV platforms
- custom drone development
